## INSIGHTS CRÍTICOS EXTRAÍDOS

### 1. Demanda = 4 Pilares Estruturados
```
DEMANDA = Comportamento de Mercado
        + Histórico de Vendas
        + Análise de Concorrência
        + Oferta Disponível
```
**Implicação:** Não é suficiente "achar que vende". Validação data-driven reduz 70% do risco.

### 2. Sazonalidade Gera 30-50% do Volume Anual
```
CRÍTICO:   Natal (Dez), Black Friday (Nov)        [15-20% cada]
IMPORTANTE: Dia das Crianças (Out), Dia Pais (Ago) [8-12% cada]
MODERADO:   Datas menores (Páscoa, Mães, Clientes) [3-5% cada]
```
**Implicação:** Planejamento de estoque deve começar 8-12 semanas antes.

### 3. Funil de Vendas Define Tudo
```
Taxa de Conversão GLOBAL: ~1.8%
Taxa com Otimização FUNIL: 3x a 5x maior (5.4% a 9%)
Elemento Crítico: Identificar gargalo e eliminar
```
**Implicação:** Investir em otimização de funil > investir em mais tráfego.

### 4. Categorias em Crescimento Exponencial
```
FITNESS:    3.5B (2024) → 16B projetado (2025+)  [Crescimento 450%+]
ELETRÔNICOS: Estável, perene, suporta scalabilidade
PET SHOP:   Crescimento consistente, alta retenção
COZINHA:    Emergente (pós-pandemia), menos saturation
```
**Implicação:** Fitness é boca-cheia em 2025, mas cozinha tem menos concorrência.

### 5. Análise de Dados é Diferenciador Crítico
```
AMADORES:      Intuição + tentativa/erro
PROFISSIONAIS: Dashboard + Análise Contínua + IA
DIFERENÇA:     60-80% melhoria em taxa de conversão
```
**Implicação:** Implementar análise de dados é investimento, não custo.

### 6. Reputação >4.5 Estrelas é Critério de Sobrevivência
```
<4.0 estrelas: Risco de suspensão de conta
4.0-4.5:       Permitido, mas com restrições
>4.5:          Acesso a todas as funcionalidades
4.8+:          Destaque nos algoritmos ML
```
**Implicação:** Compliance e qualidade valem MAIS que volume.

### 7. Produtos Chineses Têm Margem 100%+, Mas Risco Alto
```
VANTAGEM:  Custo 80% menor, margem 100-200%
RISCO:     Logística 30-60 dias, câmbio volátil, qualidade
VALIDAÇÃO: Essencial antes de pedir grande volume
```
**Implicação:** China é viável, mas precisa gestão ativa.

### 8. Estoque é Tático, não Operacional
```
FALTA:   Perda de venda (pode ser +50% loss em picos)
EXCESSO: Capital parado (10-20% a.a. em carrying cost)
IDEAL:   Rotação 4-6x/ano (varia por categoria)
```
**Implicação:** Planejamento de estoque é decisão estratégica.

```
POST   /items                  - Criar anúncio
GET    /items/{id}            - Obter detalhes
PUT    /items/{id}            - Atualizar anúncio
PUT    /items/{id}/price      - Atualizar preço
```

### Analytics (Recomendado)
```
GET    /users/{id}/items/search    - Listar anúncios do seller
GET    /orders/search              - Obter vendas com filtro
GET    /items/{id}/visits          - Visitas do anúncio
GET    /items/{id}/sales           - Histórico de vendas
```

### Promoções (Estratégico)
